## What it is (as-built)

A single-page app (`index.html`) plus a tiny local relay (`proxy.js`). Claude is the tutor; the page is the map, launcher, tracker, and record.

- **The Eval Codex UI** — a gamified 13-node skill tree of the AI-eval canon. Each node: status (`todo / gap / shaky / solid`), canonical anchor (a real source to grade against), the loop steps. Rank climbs *Vibes-Based Novice → Leading Expert*. Light/dark toggle. Atkinson Hyperlegible body + Fraunces display.
- **In-app Claude tutor** — clicking a node or "Start today's session" opens a chat that runs the tutor protocol. Powered by the Anthropic API.
- **Two run modes:**
  - **Proxy (recommended):** `node proxy.js` → open `http://127.0.0.1:8791`. The key is read from the `ANTHROPIC_LEARN_AI_EVAL_TUTOR` env var server-side and never enters the browser. Proxy binds to `127.0.0.1` only and is not a generic relay. (Verified this session: serves the app on 8791, a spoofed `Host` header is rejected with 403, and a malformed POST returns 400 with no upstream call.)
  - **file:// fallback:** open `index.html` directly; paste a spend-capped key into ⚙ (stored in browser localStorage).
- **Metalearning engine** (evidence-based): spaced-review scheduler (1/3/7/16/35-day expanding intervals, "due" queue + decay), interleaved daily session (new node + due reviews mixed), tutor v2 protocol (predict-your-score → teach with 2 examples → capture in your words → cold quiz → grade vs notes AND anchor → mock-interview transfer).
- **Audit trail + learning history** — every session, message, status change, and review is appended to `history/events.jsonl` with a readable per-session transcript in `history/transcripts/`. The 🕘 viewer browses it; "export .md" downloads it. Local-disk only, gitignored.

## Known caveats (accepted at freeze)

- **Audit trail not yet runtime-verified by the assistant** (a process-spawn test was declined). The first real session is its verification — see hardening below.
- **Model defaults to whatever's saved;** the user's localStorage currently holds Opus 4.7 (pricier). Switch to Sonnet 4.6 in ⚙ for cheaper, plenty-sharp tutoring.
- **MCP-server disruption earlier** (a blanket `node` kill) was unrelated to this app and is fixed by restarting Claude Code; it touched no files.
- **Failure behavior (sad path):** if a tutor call fails (missing/invalid key, proxy down, network), the chat shows an inline error and no assistant reply is logged — your sent message is already logged, so a session may show a user turn with no reply. Status and review events log independently of the API. No data corruption either way.
